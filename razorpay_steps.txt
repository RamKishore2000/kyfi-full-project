RAZORPAY LIVE PAYMENT KEYS - CLIENT STEPS

Purpose:
These steps are for getting Razorpay production payment credentials for KYFI subscription payments.

Important:
Do not send these keys in public chat, WhatsApp groups, screenshots, or email threads with many people.
Send the keys only to the developer through a secure/private channel.


1. Create or log in to Razorpay account

1. Open:
   https://dashboard.razorpay.com

2. Log in using the business/company Razorpay account.

3. If the account is new, complete business onboarding and KYC.

4. Razorpay live payments will work only after the account/KYC is approved by Razorpay.


2. Complete business/KYC activation

1. In Razorpay Dashboard, complete all required business details.

2. Add business name, owner details, PAN/GST details, bank account, address, and website/app details if Razorpay asks.

3. Submit documents for verification.

4. Wait until Razorpay account status is live/activated.

5. If Razorpay shows account under review, wait for approval or contact Razorpay support.


3. Generate Live API Keys

1. Open Razorpay Dashboard:
   https://dashboard.razorpay.com

2. Switch mode to Live Mode.

3. Go to:
   Settings -> API Keys

4. Click:
   Generate Key

5. Razorpay will show:
   Key Id
   Key Secret

6. Download or copy both values immediately.

7. Save the Key Secret safely. Razorpay shows the Key Secret only once.


4. Details to send to developer

Send these exact values:

RAZORPAY_KEY_ID=
RAZORPAY_KEY_SECRET=

Example format:

RAZORPAY_KEY_ID=rzp_live_xxxxxxxxxxxxxx
RAZORPAY_KEY_SECRET=xxxxxxxxxxxxxxxxxxxxxxxx


5. Optional but recommended: create webhook

Webhook is useful for payment confirmation, failed payment tracking, and future subscription/payment audit.

1. Open Razorpay Dashboard.

2. Go to:
   Settings -> Webhooks

3. Click:
   Add New Webhook

4. Webhook URL:
   https://api.kyfi.in/api/subscription/webhook

5. Create a strong webhook secret.

6. Select payment events if Razorpay asks:
   payment.captured
   payment.failed
   order.paid

7. Save webhook.

8. Send this value to developer:

RAZORPAY_WEBHOOK_SECRET=


6. Final checklist before sending credentials

Send these to developer:

1. Razorpay Live Key Id
2. Razorpay Live Key Secret
3. Webhook Secret, if webhook is created
4. Confirmation that Razorpay account is live/activated


7. Security rules

1. Do not share Key Secret publicly.

2. Do not commit keys into GitHub.

3. Do not put keys in frontend code.

4. Keys must be stored only in backend server .env file.

5. If keys are leaked, regenerate keys immediately from Razorpay Dashboard.
